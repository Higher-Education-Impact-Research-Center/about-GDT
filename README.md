# about-GDT  关于 GDT

> GDT高校影响力研究中心：为影响力决策-提供关键信息


高校影响力研究中心是广州南方学院新成立的校级研究中心，旨在参考联合国学术影响（UNAI）倡议[[1]](#1)及泰晤士高等教育THE影响力排名的研究方法[[2]](#2)，推进广州南方学院在联合国可持续发展目标的成果，最终为应用型高校在国家碳达峰与碳中和、共同富裕等高质量发展做出有真实影响的成果。
